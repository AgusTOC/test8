# Website Sederhana - Toko Online 🛍️

Sebuah contoh website e-commerce sederhana yang dibuat dengan HTML, CSS, dan JavaScript.

## 📋 Fitur-Fitur

✅ **Responsive Design** - Website responsif untuk mobile, tablet, dan desktop
✅ **Navbar Sticky** - Menu navigasi yang tetap berada di atas saat scroll
✅ **Hero Section** - Bagian utama dengan call-to-action button
✅ **Produk Grid** - Daftar produk dalam format grid yang rapi
✅ **Smooth Scroll** - Scroll halus saat navigasi antar section
✅ **Form Kontak** - Form untuk menghubungi customer service
✅ **Gradient Background** - Warna gradien modern yang menarik
✅ **Hover Effects** - Efek interaktif saat mouse hover

## 📁 Struktur File

```
toc_index.html       # File HTML utama
toc_styles.css       # File CSS untuk styling
toc_script.js        # File JavaScript untuk interaktivitas
toc_README.md        # File dokumentasi (ini)
```

## 🚀 Cara Menggunakan

1. **Buka file HTML**
   - Buka `toc_index.html` di browser Anda
   - Browser apapun bisa digunakan (Chrome, Firefox, Safari, Edge, dll)

2. **Navigasi Website**
   - Gunakan menu di atas untuk navigasi ke berbagai section
   - Klik tombol "Lihat Produk" untuk langsung ke bagian produk
   - Scroll halus otomatis saat navigasi

3. **Interaksi**
   - Klik tombol "Beli Sekarang" untuk menambah produk
   - Isi form kontak untuk mengirim pesan
   - Hover pada kartu produk untuk melihat efek visual

## 🔧 Fungsionalitas JavaScript

### scrollToSection(sectionId)
Menggerakkan halaman ke section tertentu dengan smooth scroll

### beliProduk(namaProduk)
Menampilkan notifikasi ketika produk ditambahkan ke keranjang

### submitForm(event)
Menangani pengiriman form kontak dan validasi data

### Active Link Detection
Secara otomatis menandai link aktif saat scroll ke section tertentu
